# FCE-Net_code
FCE-Net is an original real time image enhancement network. The weight obtained based on ACE algorithm can be applied to most biomedical image data sets.
# Dependencies
Python 3.7
PyTorch >= 1.8.1 + cu111
Torchvision >= 0.9.1 + cucu111
Torchsummary 1.5.1
tifffile 2020.12.8
scikit-image 0.19.2
matplotlib 3.3.2
numpy 1.21.5
Pillow 9.1.0
